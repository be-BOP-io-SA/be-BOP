# Documentação de Gestão de Temas

Acessível via **Admin** > **Merch** > **Theme**, esta interface permite gerir os temas do seu be-BOP. Permite:

- Ao proprietário da loja modificar famílias de fontes e cores através da interface de administração.
- Ao proprietário criar diferentes estilos, importá-los/exportá-los e alternar facilmente entre eles através da interface de administração.

  ![image](https://github.com/user-attachments/assets/a9d65a4b-24fa-438c-81ae-b50f12bfaaf4)

## Adicionar um Tema

Para adicionar um tema, clique em **Create a new theme**.

![image](https://github.com/user-attachments/assets/8e732d4e-aabd-4680-b043-8e1907948401)
