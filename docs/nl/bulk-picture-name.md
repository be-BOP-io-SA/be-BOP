# Configuratie Bulk Picture Name

Toegankelijk via Admin > Merch > Pictures **Bulk picture name editor**, in deze sectie kunt u de namen van afbeeldingen die zijn opgeslagen in uw S3 be-BOP in bulk wijzigen.

![image](https://github.com/user-attachments/assets/50efc726-cc53-4a45-be97-18ca9d4ac149)

## Formuliervelden

Het formulier bevat de volgende velden:

**Current slug**: De identificatie van de afbeelding (niet bewerkbaar).

**Name**: De huidige naam van de afbeelding.
