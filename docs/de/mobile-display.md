# Konfiguration der mobilen Anzeige

Zugänglich über Admin > Merch > Layout, dieser Bereich ermöglicht es Ihnen, das Designverhalten Ihres be-BOP auf mobilen Geräten zu ändern.

![image](https://github.com/user-attachments/assets/0939e4d4-6750-4750-844e-1381676d989d)

## Standardkonfiguration

Standardmäßig verwendet Ihr be-BOP diese Konfiguration:

```markdown
meta name="viewport" content="width=1000"
```

Dies ermöglicht es, das gleiche Design auf jedem Gerät beizubehalten.

## Viewport-Breite

![image](https://github.com/user-attachments/assets/054702cf-5a70-444a-8e19-d0b6f23c8d2f)

Ermöglicht es Ihnen, einen Wert für die Viewport-Breite festzulegen.

## Gerätebreite

![image](https://github.com/user-attachments/assets/7509aebb-8d6f-4b69-b2c2-d1d034d813e6)

Ermöglicht es Ihnen, `content="width=device-width"` auf bestimmte Benutzer Ihres be-BOP anzuwenden.

## CMS-Mobilzone

![image](https://github.com/user-attachments/assets/9cb296a3-d164-48b5-8a39-38f7c31c4edf)

Wenn aktiviert, werden zusätzliche CMS-Zonen (product-top, order-top...) auf Mobilgeräten ausgeblendet.
