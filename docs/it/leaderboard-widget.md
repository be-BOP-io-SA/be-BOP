# Documentazione Widget Leaderboard

Accessibile su **Admin** > **Widgets** > **Leaderboards**, questa sezione permette di visualizzare una classifica nel vostro be-BOP.

![image](https://github.com/user-attachments/assets/d9de6d66-d5ac-4f10-9d26-ab1a923946a4)

## Aggiungere un leaderboard

Per aggiungere un leaderboard cliccate su **add leaderboard**

### 1. **Leaderboard name**

Il nome del leaderboard. Un titolo breve e descrittivo che riassume l'obiettivo del leaderboard.

### 2. **Mode**

La modalità del leaderboard che può essere un numero di articoli venduti (**TotalProducts**) o una somma di denaro raccolta (**MoneyAmount**)

### 3. Currency

Un campo **currency** se la modalità è **MoneyAmount** che rappresenta la valuta. Utilizzato per la visualizzazione.

### 4. **Beginning date**

La data in cui il leaderboard inizia.

### 5. **Ending date**

La data limite di fine del leaderboard.

### 6. **Products**

Una lista di prodotti la cui vendita fa avanzare il leaderboard.

## Integrazione leaderboard nel CMS

Per aggiungere un leaderboard in una pagina o zona CMS potete utilizzare `[Leaderboard=slug]`
