# Mobile Display Configuration

Accessible via Admin > Merch > Layout, this section allows you to change the design behavior of your be-BOP on mobile devices.

![image](https://github.com/user-attachments/assets/0939e4d4-6750-4750-844e-1381676d989d)

## Default Configuration

By default, your be-BOP uses this configuration:

```markdown
meta name="viewport" content="width=1000"
```

This allows it to keep the same design on any device.

## Viewport Width

![image](https://github.com/user-attachments/assets/054702cf-5a70-444a-8e19-d0b6f23c8d2f)

Allows you to set a value for the viewport width.

## Device Width

![image](https://github.com/user-attachments/assets/7509aebb-8d6f-4b69-b2c2-d1d034d813e6)

Allows you to apply `content="width=device-width"` to specific users of your be-BOP.

## CMS Mobile Zone

![image](https://github.com/user-attachments/assets/9cb296a3-d164-48b5-8a39-38f7c31c4edf)

When checked, hides additional CMS zones (product-top, order-top...) on mobile.
