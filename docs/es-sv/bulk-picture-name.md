# Configuración masiva de nombres de imágenes

Accesible a través de Admin > Merch > Pictures **Bulk picture name editor**, esta sección te permite renombrar imágenes de forma masiva guardadas en tu almacenamiento S3 de be-BOP.

![image](https://github.com/user-attachments/assets/50efc726-cc53-4a45-be97-18ca9d4ac149)

## Campos del formulario

El formulario contiene los siguientes campos:

**Current slug**: El identificador de la imagen (no editable).

**Name**: El nombre actual de la imagen.
