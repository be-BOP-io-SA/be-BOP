# Configurazione Layout

Accessibile su **Admin** > **Merch** > **Layout**, questa sezione permette di gestire il layout del vostro sito (menu, logo, footer).

## Top Bar

![image](https://github.com/user-attachments/assets/750a0a59-8959-4413-a2e1-3283bb70b4cb)

Rappresenta la barra di menu principale del vostro be-BOP. Contiene i seguenti parametri.

### Brand Name

Un campo per cambiare il nome della vostra azienda
![image](https://github.com/user-attachments/assets/6d5f6255-0691-47b0-8edc-3e1e21839d4d)

### Logo e favicon

Link che vi permettono di cambiare il logo e la favicon del vostro be-BOP.

- **Cambiare il logo**: Andate in **pictures**, aggiungete un'immagine e definitela come logo.
- **Cambiare la favicon**: Andate in **pictures**, aggiungete un'immagine e definitela come favicon.

### Website Title

Il titolo che verrà visualizzato nella scheda di navigazione del vostro sito

### Website Description

Una descrizione del vostro sito. Viene utilizzata anche per la descrizione durante l'indicizzazione del vostro sito

![image](https://github.com/user-attachments/assets/156514a5-56f0-4e80-b482-c1c623a58151)

### Links

Menu che potete aggiungere alla vostra top bar be-BOP.

per impostazione predefinita:

| Text  | URL      |
| ----- | -------- |
| Blog  | `/blog`  |
| Store | `/store` |
| Admin | `/admin` |

**[Add topbar link]**: un pulsante per aggiungere menu

## Nav Bar

![image](https://github.com/user-attachments/assets/b5e808fe-3b97-4fa2-a64f-dfce02e8c7d7)

Rappresenta la navigazione secondaria del vostro sito. Contiene i seguenti parametri.

### Links

Sotto-menu del vostro be-BOP.
Per impostazione predefinita:

| Text       | URL           |
| ---------- | ------------- |
| Challenges | `/challenges` |
| Rewards    | `/rewards`    |
| Orders     | `/orders`     |

**[Add navbar link]**: un pulsante per aggiungere sotto-menu

---

## Footer

![image](https://github.com/user-attachments/assets/389f4404-f021-4173-8058-06ddcb1802f0)

Rappresenta la sezione footer del vostro sito. Contiene i seguenti parametri.

### Opzioni

- [ ] Display "Powered by be-BOP": visualizza un "Powered by be-BOP" nel vostro footer.
- [x] Display company identity & company contact: visualizza l'identità della vostra azienda e le vostre informazioni di contatto.
- [x] Display main shop informations: visualizza le informazioni del vostro negozio principale.

### Links

Permette di aggiungere link nel vostro footer.
Per impostazione predefinita:

| Text             | URL        |
| ---------------- | ---------- |
| Terms of Service | `/terms`   |
| Privacy Policy   | `/privacy` |

**[Add footer link]**: un pulsante per aggiungere altri link nel footer.
