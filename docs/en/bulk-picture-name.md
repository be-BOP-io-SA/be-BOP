# Bulk Picture Name Configuration

Accessible via Admin > Merch > Pictures **Bulk picture name editor**, this section allows you to bulk rename images saved in your be-BOP S3 storage.

![image](https://github.com/user-attachments/assets/50efc726-cc53-4a45-be97-18ca9d4ac149)

## Form Fields

The form contains the following fields:

**Current slug**: The image identifier (not editable).

**Name**: The current name of the image.
