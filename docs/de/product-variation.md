# Dokumentation der Produktvarianten

![image](https://github.com/user-attachments/assets/5d44dd81-a1d8-474e-b627-d5f3e01e955f)

Zugänglich beim Erstellen oder Bearbeiten eines Produkts, ermöglichen Varianten eine Auswahl (bei Farbe, Größe usw.) eines Artikels vor dem Hinzufügen zum Warenkorb.

## Erstellen oder Hinzufügen eines Produkts mit Varianten

Beim Hinzufügen oder Bearbeiten eines Produkts können Sie Varianten hinzufügen. Gehen Sie dazu wie folgt vor:

- Aktivieren Sie **This is a standalone product**, damit das Produkt mit Varianten nicht mit einer Menge größer als 1 zum Warenkorb hinzugefügt wird.
- Aktivieren Sie **Product has light variations (no stock difference)**, um verschiedene Varianten hinzufügen zu können.

  ![image](https://github.com/user-attachments/assets/80bfe14e-cec9-4b29-b8e2-eb3067a29b26)

Um eine Variante hinzuzufügen, füllen Sie aus:

- **Name**: Der Titel der Variante (Beispiel: Größe)
- **Value**: Ein Wert für den Namen (Beispiel: XL)
- **Price difference**: Der Zahlenwert (in der gleichen Währung wie der Produktpreis), der hinzugefügt wird, wenn ein Benutzer diese Variante wählt.
  (Beispiel: ein XL-T-Shirt kostet 2 Euro mehr als der Grundpreis eines T-Shirts)

Klicken Sie auf **add variation**, um eine neue Variante hinzuzufügen. Das Symbol '🗑️' ermöglicht das Löschen einer bereits in der Datenbank gespeicherten Variante.

## Anzeige auf der Produktseite

![image](https://github.com/user-attachments/assets/ed13cc76-330b-4c3c-b162-52f6438ccca3)

Wenn Sie ein Produkt mit Varianten anzeigen, können Sie die Varianten vor dem Hinzufügen zum Warenkorb auswählen. Der Name und die Variantenliste sind auf der Produktseite zugänglich.

## Anzeige im Warenkorb

![image](https://github.com/user-attachments/assets/747aed2c-854f-4bf4-a156-9a1b18f3616e)

![image](https://github.com/user-attachments/assets/394264ed-91c0-478c-9aa1-b9064c7c1b6b)

Wenn ein Produkt mit Varianten zum Warenkorb hinzugefügt wird, wird der Name zum Produktnamen zusammengesetzt mit dem Variantenwert.

Beispiel: T-Shirt - XL - Rot (wobei XL ein Variantenwert von Größe und Rot ein Variantenwert von Farbe ist)
