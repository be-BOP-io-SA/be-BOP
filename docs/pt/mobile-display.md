# Configuração de Exibição Móvel

Acessível via Admin > Merch > Layout, esta secção permite alterar o comportamento do design do seu be-BOP em dispositivos móveis.

![image](https://github.com/user-attachments/assets/0939e4d4-6750-4750-844e-1381676d989d)

## Configuração Padrão

Por defeito, o seu be-BOP usa esta configuração:

```markdown
meta name="viewport" content="width=1000"
```

Isto permite manter o mesmo design em qualquer dispositivo.

## Largura do Viewport

![image](https://github.com/user-attachments/assets/054702cf-5a70-444a-8e19-d0b6f23c8d2f)

Permite definir um valor para a largura do viewport.

## Largura do Dispositivo

![image](https://github.com/user-attachments/assets/7509aebb-8d6f-4b69-b2c2-d1d034d813e6)

Permite aplicar `content="width=device-width"` a utilizadores específicos do seu be-BOP.

## Zona CMS Móvel

![image](https://github.com/user-attachments/assets/9cb296a3-d164-48b5-8a39-38f7c31c4edf)

Quando marcado, oculta zonas CMS adicionais (product-top, order-top...) no telemóvel.
