# Configuratie Mobiele Weergave

Toegankelijk via Admin > Merch > Layout, in deze sectie kunt u het gedrag van het ontwerp van uw be-BOP op mobiel wijzigen.

![image](https://github.com/user-attachments/assets/0939e4d4-6750-4750-844e-1381676d989d)

## Standaardconfiguratie

Standaard gebruikt uw be-BOP deze configuratie

```markdown
meta name="viewport" content="width=1000"
```

Hierdoor behoudt het hetzelfde ontwerp op elk apparaat.

## Viewport width

![image](https://github.com/user-attachments/assets/054702cf-5a70-444a-8e19-d0b6f23c8d2f)

Stelt u in staat een waarde toe te kennen aan de viewport width.

## Device width

![image](https://github.com/user-attachments/assets/7509aebb-8d6f-4b69-b2c2-d1d034d813e6)

Maakt het mogelijk om `content="width=device-width"` toe te passen voor specifieke gebruikers van uw be-BOP.

## CMS Zone Mobiel

![image](https://github.com/user-attachments/assets/9cb296a3-d164-48b5-8a39-38f7c31c4edf)

Wanneer aangevinkt, worden de aanvullende CMS-zones (product-top, order-top....) niet weergegeven op mobiel.
