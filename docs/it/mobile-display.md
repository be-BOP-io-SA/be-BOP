# Configurazione Display Mobile

Accessibile su Admin > Merch > Layout, questa sezione permette di cambiare il comportamento del design del vostro be-BOP su mobile.

![image](https://github.com/user-attachments/assets/0939e4d4-6750-4750-844e-1381676d989d)

## Configurazione predefinita

Per impostazione predefinita il vostro be-BOP utilizza questa configurazione

```markdown
meta name="viewport" content="width=1000"
```

Che gli permette di mantenere lo stesso design su qualsiasi dispositivo

## Viewport width

![image](https://github.com/user-attachments/assets/054702cf-5a70-444a-8e19-d0b6f23c8d2f)

Permette di assegnare un valore al viewport width

## Device width

![image](https://github.com/user-attachments/assets/7509aebb-8d6f-4b69-b2c2-d1d034d813e6)

Permette di applicare `content="width=device-width"` a utenti specifici del vostro be-BOP.

## CMS Zone Mobile

![image](https://github.com/user-attachments/assets/9cb296a3-d164-48b5-8a39-38f7c31c4edf)

Quando è selezionato, non visualizza le zone CMS aggiuntive (product-top, order-top....) su mobile.
