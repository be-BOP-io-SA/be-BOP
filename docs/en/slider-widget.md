# Slider Widget Documentation

Available via **Admin** > **Widgets** > **Slider**, this interface allows you to create a slider (image carousel) used to present attractive visual content on your be-BOP.

![image](https://github.com/user-attachments/assets/b738d8ae-edd8-4d0f-b33c-72ea8f2311bd)

---

## Adding a Slider

To add a slider, click on **Add slider**.

![image](https://github.com/user-attachments/assets/75483ff8-032d-487c-92e7-4b38068d1d8c)

### 1. **Slider Title**

- **Description**: The slider title. This text can be used as a reference to identify the slider in the system.
- **Usage**: Optional, but useful for organizing and managing sliders.

### 2. **Slug**

- **Description**: Unique identifier for the slider, used in URLs or as an access key.
- **Constraints**:
  - Can only contain lowercase letters, numbers, and hyphens (`-`).

### 3. **Picture**

- **Description**: Main image to display in the slider.
- **Functionality**:

  - A field to browse and upload a file from your computer.
  - Only accepts image formats (JPG, PNG, etc.).

![image](https://github.com/user-attachments/assets/1cf9c9c4-4e19-4093-b5cb-1b1d88ffaf88)

## Editing a Slider

When you click on a slider title, you can edit it by adding new images.

![image](https://github.com/user-attachments/assets/226847f0-3c05-4b8a-8620-8a42015444f6)

### 1. **Add Picture**

Clicking **Add picture** will take you to a new page to associate images with your slider.

![image](https://github.com/user-attachments/assets/dee5e891-e9d6-4849-b04c-926a33ebbd6b)

Each slide can be associated with a URL that will open when you click on the slide. And an **Open in new tab** option to open the URL in a new window.

![image](https://github.com/user-attachments/assets/a57e06b1-6a64-4311-a9c3-5458cfcda1bc)

## CMS Integration

To integrate your slider into a CMS zone or page, add it as follows: `[Slider=slug]`.
