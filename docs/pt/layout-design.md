# Configuração do Layout

Acessível via **Admin** > **Merch** > **Layout**, esta secção permite gerir o layout do seu site (menus, logo, rodapé).

## Barra Superior

![image](https://github.com/user-attachments/assets/750a0a59-8959-4413-a2e1-3283bb70b4cb)

Representa a barra de menu principal do seu be-BOP. Contém as seguintes configurações.

### Nome da Marca

Um campo para alterar o nome da sua empresa.
![image](https://github.com/user-attachments/assets/6d5f6255-0691-47b0-8edc-3e1e21839d4d)

### Logo e Favicon

Links que permitem alterar o logo e o favicon do seu be-BOP.

- **Alterar o logo**: Vá a **pictures**, adicione uma imagem e defina-a como logo.
- **Alterar o favicon**: Vá a **pictures**, adicione uma imagem e defina-a como favicon.

### Título do Website

O título que será exibido no separador do navegador do seu site.

### Descrição do Website

Uma descrição do seu site. Também usada para a descrição na indexação do seu site para SEO.

![image](https://github.com/user-attachments/assets/156514a5-56f0-4e80-b482-c1c623a58151)

### Links

Menus que pode adicionar à barra superior do seu be-BOP.

Por defeito:

| Text  | URL      |
| ----- | -------- |
| Blog  | `/blog`  |
| Store | `/store` |
| Admin | `/admin` |

**[Add topbar link]**: Um botão para adicionar mais menus.

## Barra de Navegação

![image](https://github.com/user-attachments/assets/b5e808fe-3b97-4fa2-a64f-dfce02e8c7d7)

Representa a navegação secundária do seu site. Contém as seguintes configurações.

### Links

Submenus do seu be-BOP.
Por defeito:

| Text       | URL           |
| ---------- | ------------- |
| Challenges | `/challenges` |
| Rewards    | `/rewards`    |
| Orders     | `/orders`     |

**[Add navbar link]**: Um botão para adicionar mais submenus.

---

## Rodapé

![image](https://github.com/user-attachments/assets/389f4404-f021-4173-8058-06ddcb1802f0)

Representa a secção de rodapé do seu site. Contém as seguintes configurações.

### Opções

- [ ] Display "Powered by be-BOP": Exibe um "Powered by be-BOP" no seu rodapé.
- [x] Display company identity & company contact: Exibe a identidade da sua empresa e informações de contacto.
- [x] Display main shop informations: Exibe as informações da sua loja principal.

### Links

Links no seu rodapé.
Por defeito:

| Text             | URL        |
| ---------------- | ---------- |
| Terms of Service | `/terms`   |
| Privacy Policy   | `/privacy` |

**[Add footer link]**: Um botão para adicionar mais links no rodapé.
