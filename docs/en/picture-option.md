# Logo & Favicon Configuration

![image](https://github.com/user-attachments/assets/31ac6500-4858-4dd3-a4b3-bb3898dd1125)

Available via **Admin** > **Merch** > **Pictures** > **Image**, this page allows you to manage image settings and usage as logo or favicon in your be-BOP application.

---

## **Adding an Image**

To add an image, go to **Admin** > **Merch** > **Pictures**, then click **New picture**.

![image](https://github.com/user-attachments/assets/52a583c8-e9a9-4f8b-aa2d-f295a29b9b91)

After saving, you can access the image options by clicking on it.

![image](https://github.com/user-attachments/assets/c6c09f64-f30f-40df-acee-7bc9a61b6c06)

## **Management Options**

### **1. Image will be a wide logo**

Check this box if the logo is in a wide format. Enable it if your logo needs to be adapted for horizontal display.

---

### **2. Set as logo**

Replaced by a **Remove from logo** button (when the image is already set as logo), this option allows you to set the current image as the default logo.

---

### **3. Set as dark logo**

Replaced by a **Remove from dark logo** button (when the image is already set as the dark theme logo), this option allows you to set the image as the logo to use for dark themes.

---

### **4. Set as footer logo**

Replaced by a **Remove from footer logo** button (when the image is already set as the footer logo), this option allows you to set the image as the logo to display in the footer.

---

### **5. Set as favicon**

Replaced by a **Remove from favicon** button (when the image is already set as favicon), this option allows you to set the current image as the favicon.

---
