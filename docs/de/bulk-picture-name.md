# Massenbearbeitung von Bildnamen

Zugänglich über Admin > Merch > Pictures **Bulk picture name editor**, ermöglicht Ihnen dieser Bereich die Massenumbenennung von Bildern, die in Ihrem be-BOP S3-Speicher gespeichert sind.

![image](https://github.com/user-attachments/assets/50efc726-cc53-4a45-be97-18ca9d4ac149)

## Formularfelder

Das Formular enthält die folgenden Felder:

**Current slug**: Die Bildkennung (nicht bearbeitbar).

**Name**: Der aktuelle Name des Bildes.
