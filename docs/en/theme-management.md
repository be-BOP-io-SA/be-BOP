# Theme Management Documentation

Accessible via **Admin** > **Merch** > **Theme**, this interface allows you to manage themes on your be-BOP. It allows:

- The store owner to modify font families and colors via the admin UI.
- The owner to create different styles, import/export them, and easily switch between them via the admin UI.

  ![image](https://github.com/user-attachments/assets/a9d65a4b-24fa-438c-81ae-b50f12bfaaf4)

## Adding a Theme

To add a theme, click on **Create a new theme**.

![image](https://github.com/user-attachments/assets/8e732d4e-aabd-4680-b043-8e1907948401)

In **Admin** > **Merch** > **Themes**, you can choose the created theme as the theme for your be-BOP.
