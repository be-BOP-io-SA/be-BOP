# Configuración de Logo y Favicon

![image](https://github.com/user-attachments/assets/31ac6500-4858-4dd3-a4b3-bb3898dd1125)

Disponible a través de **Admin** > **Merch** > **Pictures** > **Image**, esta página permite gestionar la configuración de imágenes y su uso como logo o favicon en su aplicación be-BOP.

---

## **Agregar una Imagen**

Para agregar una imagen, vaya a **Admin** > **Merch** > **Pictures** y haga clic en **New picture**.

![image](https://github.com/user-attachments/assets/52a583c8-e9a9-4f8b-aa2d-f295a29b9b91)

Después de guardar, puede acceder a las opciones de la imagen haciendo clic en ella.

![image](https://github.com/user-attachments/assets/c6c09f64-f30f-40df-acee-7bc9a61b6c06)

## **Opciones de Gestión**

### **1. Image will be a wide logo**
Marque esta casilla si el logo tiene formato ancho. Actívelo si su logo necesita adaptarse para visualización horizontal.

### **2. Set as logo**
Reemplazado por un botón **Remove from logo** (cuando la imagen ya está definida como logo), esta opción permite definir la imagen actual como logo predeterminado.

### **3. Set as dark logo**
Reemplazado por un botón **Remove from dark logo**, esta opción permite definir la imagen como logo para temas oscuros.

### **4. Set as footer logo**
Reemplazado por un botón **Remove from footer logo**, esta opción permite definir la imagen como logo del footer.

### **5. Set as favicon**
Reemplazado por un botón **Remove from favicon**, esta opción permite definir la imagen actual como favicon.

## **Acciones**

### **1. Update**
![image](https://github.com/user-attachments/assets/d2f774d4-aab2-450b-be47-732c5e73a92c)
- **Descripción**: Guarda los cambios realizados al cambiar el nombre de la imagen.

### **2. Delete**
- **Descripción**: Elimina permanentemente la imagen seleccionada.

## **Volver a la Lista**
- Use el enlace **Back to list** para volver a la lista de imágenes disponibles.
