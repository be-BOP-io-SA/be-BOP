# Galerie-Widget-Dokumentation

Verfügbar über **Admin** > **Widgets** > **Gallery**, Galerie-Widgets können in Ihrem be-BOP verwendet werden, um 3-Bilder-Galerien in CMS-Zonen oder -Seiten zu integrieren.

![image](https://github.com/user-attachments/assets/77697e24-f90c-4e6e-828a-235b99da1d34)

## Eine Galerie hinzufügen

Um eine Galerie hinzuzufügen, klicken Sie auf **Add contact form**.

![image](https://github.com/user-attachments/assets/9730f949-57a2-4508-aff9-edc3edcfa84c)

Die Oberfläche **"Add a gallery"** ermöglicht es Benutzern, eine neue Galerie mit spezifischen Informationen hinzuzufügen, wie Name, Slug, Hauptinhalt und Sekundärinhalt.

### Formularfelder

- **Gallery Name**:
  Textfeld zur Definition des Galerienamens.
  _Beispiel: "Herbstgalerie"_

- **Gallery Slug**:
  Eindeutiger Galeriebezeichner, der in der URL verwendet wird, für CMS-Integration...
  _Beispiel: "herbst-galerie"_

- **Hauptgalerie**:

  ![image](https://github.com/user-attachments/assets/6c5e6376-7150-44b0-9c8b-dbf9cde280eb)

  - **Gallery Title**: Haupttitel der Galerie.
    _Beispiel: "Beste Fotogalerie"_
  - **Gallery Content**: Textfeld zum Hinzufügen von Beschreibungen oder spezifischen Informationen zur Galerie.

- **Zugehöriger Button**:

  - **Text**: Beschreibung des Buttons der Galerie.
    _Beispiel: "Mehr anzeigen"_
  - **URL**: URL-Link, der auf eine Seite oder zusätzliche Inhalte verweist, wenn der Button geklickt wird.
  - **Open in new tab**: Option, den Link in einem neuen Browser-Tab zu öffnen.

- **Sekundärgalerie** (es gibt 3):

  ![image](https://github.com/user-attachments/assets/282ee84b-fbc3-4c75-b443-0e1fcd4afe7e)

  - **Gallery subtitle**: Titel dieser Sekundärgalerie.
    _Beispiel: "Beste Fotogalerie"_
  - **Gallery subcontent**: Textfeld zum Hinzufügen von Beschreibungen oder spezifischen Informationen zu dieser Sekundärgalerie.

- **Zugehöriges Bild**: Ein Bild, das dieser Sekundärgalerie zugeordnet wird.

- **Zugehöriger Button**:

  - **Text**: Beschreibung des Buttons dieser Sekundärgalerie.
    _Beispiel: "Mehr anzeigen"_
  - **URL**: URL-Link, der auf eine Seite oder zusätzliche Inhalte verweist, wenn der Button geklickt wird.
  - **Open in new tab**: Option, den Link in einem neuen Browser-Tab zu öffnen.

## CMS-Integration

Um eine Galerie in eine CMS-Zone oder -Seite zu integrieren, fügen Sie sie wie folgt hinzu: `[Gallery=slug]`.

![image](https://github.com/user-attachments/assets/1ed5fa0c-05a5-4fc9-adad-cc84c871822c)

Und Ihre Galerie wird Ihren Benutzern wie folgt angezeigt.
