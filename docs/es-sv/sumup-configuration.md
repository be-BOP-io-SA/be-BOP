# Configuración de SumUp para la Aplicación be-BOP

Para integrar SumUp como método de pago en su aplicación **be-BOP**, vaya a **Admin** > **Payment Settings** > **SumUp**.

![image](https://github.com/user-attachments/assets/9eea3d50-1963-4102-a8cc-502caeb295db)

## Ajustes de Configuración

### 1. API Key
- **Descripción**: Clave API proporcionada por SumUp para autenticar transacciones.
- **Ejemplo**: `YOUR_SUMUP_API_KEY`

### 2. Merchant Code
- **Descripción**: Código único de identificación del comerciante proporcionado por SumUp.
- **Ejemplo**: `YOUR_MERCHANT_CODE`

### 3. Currency
- **Descripción**: Moneda utilizada para las transacciones SumUp.
- **Ejemplo**: `USD`

![image](https://github.com/user-attachments/assets/d79fa78e-1ec9-4f71-b19e-5cea9861f278)

## Pago con Tarjeta

Después de la configuración, ahora puede recibir pagos con tarjeta.
