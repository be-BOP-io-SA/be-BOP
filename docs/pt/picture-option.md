# Configuração de Logo & Favicon

![image](https://github.com/user-attachments/assets/31ac6500-4858-4dd3-a4b3-bb3898dd1125)

Disponível via **Admin** > **Merch** > **Pictures** > **Image**, esta página permite gerir as configurações de imagem e a sua utilização como logo ou favicon na sua aplicação be-BOP.

---

## **Adicionar uma Imagem**

Para adicionar uma imagem, vá a **Admin** > **Merch** > **Pictures** e clique em **New picture**.

![image](https://github.com/user-attachments/assets/52a583c8-e9a9-4f8b-aa2d-f295a29b9b91)

Após guardar, pode aceder às opções da imagem clicando nela.

![image](https://github.com/user-attachments/assets/c6c09f64-f30f-40df-acee-7bc9a61b6c06)

## **Opções de Gestão**

### **1. Image will be a wide logo**

Marque esta caixa se o logo estiver num formato largo. Ative se o seu logo precisar de ser adaptado para exibição horizontal.

---

### **2. Set as logo**

Substituído por um botão **Remove from logo** (quando a imagem já está definida como logo), esta opção permite definir a imagem atual como logo padrão.

---

### **3. Set as dark logo**

Substituído por um botão **Remove from dark logo** (quando a imagem já está definida como logo do tema escuro), esta opção permite definir a imagem como logo para temas escuros.

---

### **4. Set as footer logo**

Substituído por um botão **Remove from footer logo** (quando a imagem já está definida como logo do rodapé), esta opção permite definir a imagem como logo para exibir no rodapé.

---

### **5. Set as favicon**

Substituído por um botão **Remove from favicon** (quando a imagem já está definida como favicon), esta opção permite definir a imagem atual como favicon.

---

## **Ações**

### **1. Update**

![image](https://github.com/user-attachments/assets/d2f774d4-aab2-450b-be47-732c5e73a92c)

- **Descrição**: Guarda as alterações feitas ao alterar o nome da imagem.

### **2. Delete**

- **Descrição**: Elimina permanentemente a imagem selecionada.

---

## **Voltar à Lista**

- Use o link **Back to list** para voltar à lista de imagens disponíveis.
