# Gallery Widget Documentation

Available via **Admin** > **Widgets** > **Gallery**, gallery widgets can be used in your be-BOP to integrate 3-image galleries into CMS zones or pages.

![image](https://github.com/user-attachments/assets/77697e24-f90c-4e6e-828a-235b99da1d34)

## Adding a Gallery

To add a gallery, click on **Add contact form**.

![image](https://github.com/user-attachments/assets/9730f949-57a2-4508-aff9-edc3edcfa84c)

The **"Add a gallery"** interface allows users to add a new gallery with specific information, such as name, slug, main content, and secondary content.

### Form Fields

- **Gallery Name**:
  Text field to define the gallery name.
  _Example: "Autumn Gallery"_

- **Gallery Slug**:
  Unique gallery identifier used in the URL, for CMS integration...
  _Example: "autumn-gallery"_

- **Principal Gallery**:

  ![image](https://github.com/user-attachments/assets/6c5e6376-7150-44b0-9c8b-dbf9cde280eb)

  - **Gallery Title**: Main title of the gallery.
    _Example: "Best Photos Gallery"_
  - **Gallery Content**: Text field to add descriptions or specific information about the gallery.

- **Associated button**:

  - **Text**: Description of the button associated with the gallery.
    _Example: "See More"_
  - **URL**: URL link pointing to a page or additional content when clicking the button.
  - **Open in new tab**: Option to open the link in a new browser tab.

- **Secondary Gallery** (there are 3):

  ![image](https://github.com/user-attachments/assets/282ee84b-fbc3-4c75-b443-0e1fcd4afe7e)

  - **Gallery subtitle**: Title of this secondary gallery.
    _Example: "Best Photos Gallery"_
  - **Gallery subcontent**: Text field to add descriptions or specific information about this secondary gallery.

- **Associated image**: An image that will be associated with this secondary gallery.

- **Associated button**:

  - **Text**: Description of the button associated with this secondary gallery.
    _Example: "See More"_
  - **URL**: URL link pointing to a page or additional content when clicking the button.
  - **Open in new tab**: Option to open the link in a new browser tab.

## CMS Integration

To integrate a gallery into a CMS zone or page, add it as follows: `[Gallery=slug]`.

![image](https://github.com/user-attachments/assets/1ed5fa0c-05a5-4fc9-adad-cc84c871822c)
