# Documentación del Widget Leaderboard

Accesible a través de **Admin** > **Widgets** > **Leaderboards**, esta sección permite mostrar una tabla de clasificación en su be-BOP.

![image](https://github.com/user-attachments/assets/d9de6d66-d5ac-4f10-9d26-ab1a923946a4)

## Agregar un Leaderboard

Para agregar un leaderboard, haga clic en **add leaderboard**.

### 1. **Leaderboard name**

El nombre del leaderboard. Un título corto y descriptivo que resume el objetivo del leaderboard.

### 2. **Mode**

El modo del leaderboard, que puede ser un número de artículos vendidos (**TotalProducts**) o una cantidad de dinero recolectada (**MoneyAmount**).

### 3. Currency

Un campo **currency** si el modo es **MoneyAmount**, que representa la moneda. Utilizado para la visualización.

### 4. **Beginning date**

La fecha en que comienza el leaderboard.

### 5. **Ending date**

La fecha de finalización del leaderboard.

### 6. **Products**

Una lista de productos cuyas ventas contribuyen al leaderboard.

## Integración del Leaderboard en CMS

Para agregar un leaderboard a una página o zona CMS, puede usar `[Leaderboard=slug]`.
