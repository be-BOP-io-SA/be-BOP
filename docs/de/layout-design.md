# Layout-Konfiguration

Zugänglich über **Admin** > **Merch** > **Layout**, dieser Bereich ermöglicht es Ihnen, das Layout Ihrer Website zu verwalten (Menüs, Logo, Footer).

## Obere Leiste

![image](https://github.com/user-attachments/assets/750a0a59-8959-4413-a2e1-3283bb70b4cb)

Stellt die Hauptmenüleiste Ihres be-BOP dar. Sie enthält die folgenden Einstellungen.

### Markenname

Ein Feld zum Ändern Ihres Firmennamens.
![image](https://github.com/user-attachments/assets/6d5f6255-0691-47b0-8edc-3e1e21839d4d)

### Logo und Favicon

Links, mit denen Sie das Logo und das Favicon Ihres be-BOP ändern können.

- **Logo ändern**: Gehen Sie zu **pictures**, fügen Sie ein Bild hinzu und setzen Sie es als Logo.
- **Favicon ändern**: Gehen Sie zu **pictures**, fügen Sie ein Bild hinzu und setzen Sie es als Favicon.

### Website-Titel

Der Titel, der im Browser-Tab Ihrer Website angezeigt wird.

### Website-Beschreibung

Eine Beschreibung Ihrer Website. Wird auch für die Beschreibung bei der Indexierung Ihrer Website für SEO verwendet.

![image](https://github.com/user-attachments/assets/156514a5-56f0-4e80-b482-c1c623a58151)

### Links

Menüs, die Sie zur oberen Leiste Ihres be-BOP hinzufügen können.

Standardmäßig:

| Text  | URL      |
| ----- | -------- |
| Blog  | `/blog`  |
| Store | `/store` |
| Admin | `/admin` |

**[Add topbar link]**: Eine Schaltfläche zum Hinzufügen weiterer Menüs.

## Navigationsleiste

![image](https://github.com/user-attachments/assets/b5e808fe-3b97-4fa2-a64f-dfce02e8c7d7)

Stellt die sekundäre Navigation Ihrer Website dar. Sie enthält die folgenden Einstellungen.

### Links

Untermenüs Ihres be-BOP.
Standardmäßig:

| Text       | URL           |
| ---------- | ------------- |
| Challenges | `/challenges` |
| Rewards    | `/rewards`    |

**[Add navbar link]**: Eine Schaltfläche zum Hinzufügen weiterer Untermenüs.

## Footer

![image](https://github.com/user-attachments/assets/389f4404-f021-4173-8058-06ddcb1802f0)

Stellt den Footer-Bereich Ihrer Website dar. Er enthält die folgenden Einstellungen.

### Optionen

- [ ] Display "Powered by be-BOP": Zeigt ein "Powered by be-BOP" in Ihrem Footer an.
- [x] Display company identity & company contact: Zeigt Ihre Firmenidentität und Kontaktinformationen an.
- [x] Display main shop informations: Zeigt Ihre Hauptshop-Informationen an.

### Links

Links in Ihrem Footer.
Standardmäßig:

| Text             | URL        |
| ---------------- | ---------- |
| Terms of Service | `/terms`   |
| Privacy Policy   | `/privacy` |

**[Add footer link]**: Eine Schaltfläche zum Hinzufügen weiterer Footer-Links.
