# Leaderboard-Widget-Dokumentation

Zugänglich über **Admin** > **Widgets** > **Leaderboards**, dieser Bereich ermöglicht es Ihnen, ein Leaderboard in Ihrem be-BOP anzuzeigen.

![image](https://github.com/user-attachments/assets/d9de6d66-d5ac-4f10-9d26-ab1a923946a4)

## Ein Leaderboard hinzufügen

Um ein Leaderboard hinzuzufügen, klicken Sie auf **add leaderboard**.

### 1. **Leaderboard name**

Der Name des Leaderboards. Ein kurzer, beschreibender Titel, der das Ziel des Leaderboards zusammenfasst.

### 2. **Mode**

Der Leaderboard-Modus, der eine Anzahl verkaufter Artikel (**TotalProducts**) oder ein gesammelter Geldbetrag (**MoneyAmount**) sein kann.

### 3. Currency

Ein **currency**-Feld, wenn der Modus **MoneyAmount** ist, das die Währung darstellt. Wird für Anzeigezwecke verwendet.

### 4. **Beginning date**

Das Startdatum des Leaderboards.

### 5. **Ending date**

Das Enddatum des Leaderboards.

### 6. **Products**

Eine Liste von Produkten, deren Verkäufe zum Leaderboard beitragen.

## CMS-Leaderboard-Integration

Um ein Leaderboard zu einer CMS-Seite oder -Zone hinzuzufügen, verwenden Sie `[Leaderboard=slug]`.
