# Configuración de Precios de Productos en Masa

Accesible a través de Admin > Merch > Products **Products price**, esta sección permite cambiar en masa los precios de los productos disponibles.

![image](https://github.com/user-attachments/assets/e13749db-afbe-4196-8ba8-170e95a4f186)

## Campos del Formulario

Para cada producto puede cambiar:

**Price amount**: El precio actual del producto.

**Price currency**: La moneda actual del producto.
