# Leaderboard Widget Documentation

Accessible via **Admin** > **Widgets** > **Leaderboards**, this section allows you to display a leaderboard in your be-BOP.

![image](https://github.com/user-attachments/assets/d9de6d66-d5ac-4f10-9d26-ab1a923946a4)

## Adding a Leaderboard

To add a leaderboard, click on **add leaderboard**.

### 1. **Leaderboard name**

The leaderboard name. A short, descriptive title that summarizes the leaderboard objective.

### 2. **Mode**

The leaderboard mode, which can be a number of items sold (**TotalProducts**) or an amount of money collected (**MoneyAmount**).

### 3. Currency

A **currency** field if the mode is **MoneyAmount**, representing the currency. Used for display purposes.

### 4. **Beginning date**

The date the leaderboard starts.

### 5. **Ending date**

The end date of the leaderboard.

### 6. **Products**

A list of products whose sales contribute to the leaderboard.

## CMS Leaderboard Integration

To add a leaderboard to a CMS page or zone, you can use `[Leaderboard=slug]`.
