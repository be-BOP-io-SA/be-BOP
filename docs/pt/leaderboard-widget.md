# Documentação do Widget Leaderboard

Acessível via **Admin** > **Widgets** > **Leaderboards**, esta secção permite exibir um leaderboard no seu be-BOP.

![image](https://github.com/user-attachments/assets/d9de6d66-d5ac-4f10-9d26-ab1a923946a4)

## Adicionar um Leaderboard

Para adicionar um leaderboard, clique em **add leaderboard**.

### 1. **Leaderboard name**

O nome do leaderboard. Um título curto e descritivo que resume o objetivo do leaderboard.

### 2. **Mode**

O modo do leaderboard, que pode ser um número de itens vendidos (**TotalProducts**) ou um montante de dinheiro recolhido (**MoneyAmount**).

### 3. Currency

Um campo **currency** se o modo for **MoneyAmount**, representando a moeda. Usado para fins de exibição.

### 4. **Beginning date**

A data de início do leaderboard.

### 5. **Ending date**

A data de fim do leaderboard.

### 6. **Products**

Uma lista de produtos cujas vendas contribuem para o leaderboard.

## Integração CMS do Leaderboard

Para adicionar um leaderboard a uma página ou zona CMS, pode usar `[Leaderboard=slug]`.
