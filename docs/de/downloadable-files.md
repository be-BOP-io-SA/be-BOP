# Herunterladbare Dateien zu Produkten oder CMS-Seiten hinzufügen

Diese Anleitung erklärt, wie Sie herunterladbare Dateien (z.B. Produkthinweise oder Anleitungen) hinzufügen, die öffentlich zugänglich und nicht durch Tokens oder Bestellungen geschützt sind. Diese Dateien können direkt mit Produktseiten oder CMS-Seiten verknüpft werden.

## Schritt 1: Zugriff auf den Bereich **Files**

1. Melden Sie sich im Administrationsbereich an.
2. Navigieren Sie zum Abschnitt **Merch** im oberen Menü.
3. Klicken Sie auf **Files** im Untermenü.

## Schritt 2: Eine neue Datei hinzufügen

![image](https://github.com/user-attachments/assets/3b9ca426-34f1-4905-af05-24f3b16ade14)

1. Klicken Sie auf der Seite **List of file with no associated product** auf **New file**.

![image](https://github.com/user-attachments/assets/f4f64cd9-bedb-4af9-8626-90178c95b178)

2. Füllen Sie die erforderlichen Felder aus:

   - **Name of the file**: Geben Sie einen beschreibenden Namen an (z.B. "Produkthinweis").
   - **File**: Laden Sie die Datei hoch, indem Sie auf **Browse...** klicken und die entsprechende Datei von Ihrem Computer auswählen.

3. Klicken Sie auf die Schaltfläche **Add**, um die Datei hochzuladen.

![image](https://github.com/user-attachments/assets/2687f745-4c29-4c85-8c92-6df71c041b91)

4. **Verwaltung einer bestehenden Datei:**

   - Sobald die Datei hinzugefügt wurde, können Sie sie in der Dateiliste einsehen.
   - Klicken Sie auf eine Datei, um ihren Namen zu bearbeiten, einen Link zu erhalten oder sie zu löschen.
   - **Permanenter Link:**
      Sobald die Datei hinzugefügt wurde, wird ein gültiger permanenter Link generiert. Sie können ihn kopieren, indem Sie auf das "Kopieren"-Symbol neben dem Link klicken.
     Beispiel-Link:
      `http://localhost:5173/digital-file/raw/<dateiname>?key=<eindeutiger-schlüssel>`
     Sie können diesen Link in Ihren CMS-Seiten verwenden, um Ihre Dateien zu teilen.
   - **Direkt herunterladen:**
     Benutzer können auf die Schaltfläche **"Download file"** klicken, um sofort auf die Datei zuzugreifen.
   - **Eine Datei löschen**
     Wenn eine Datei nicht mehr benötigt wird, öffnen Sie sie aus der Liste und klicken Sie auf **"Delete"**. Eine Bestätigung wird vor dem endgültigen Löschen angefordert.
