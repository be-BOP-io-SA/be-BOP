# Aggiungere file scaricabili ai prodotti o alle pagine CMS

Questa guida spiega come aggiungere file scaricabili (ad esempio, istruzioni o guide prodotto) che sono accessibili pubblicamente e non sono protetti da token o ordini. Questi file possono essere collegati direttamente alle pagine prodotto o alle pagine CMS.

## Passo 1: Accedere alla sezione **Files**

1. Accedete al pannello di amministrazione.
2. Navigate verso la sezione **Merch** nel menu superiore.
3. Cliccate su **Files** nel sotto-menu.

## Passo 2: Aggiungere un nuovo file

![image](https://github.com/user-attachments/assets/3b9ca426-34f1-4905-af05-24f3b16ade14)

1. Nella pagina **List of file with no associated product**, cliccate su **New file**.

![image](https://github.com/user-attachments/assets/f4f64cd9-bedb-4af9-8626-90178c95b178)

2. Compilate i campi richiesti:

   - **Name of the file**: Fornite un nome descrittivo (ad esempio, "Istruzioni prodotto").
   - **File**: Caricate il file cliccando su **Sfoglia...** e selezionando il file appropriato dal vostro computer.

3. Cliccate sul pulsante **Add** per caricare il file.

![image](https://github.com/user-attachments/assets/2687f745-4c29-4c85-8c92-6df71c041b91)

4. **Gestire un file esistente:**

   - Una volta aggiunto il file, potete visualizzarlo nella lista dei file.
   - Cliccate su un file per modificarne il nome, ottenere un link o eliminarlo
   - **Link permanente:**
      Una volta aggiunto il file, viene generato un link permanente valido. Potete copiarlo cliccando sull'icona "Copia" situata accanto al link.
     Esempio di link:
      `http://localhost:5173/digital-file/raw/<file-name>?key=<unique-key>`
     Potete utilizzare questo link nelle vostre pagine CMS per condividere i vostri file.
   - **Scaricare direttamente:**
     Gli utenti possono cliccare sul pulsante **"Download file"** per accedere immediatamente al file.
   - **Eliminare un file**
     Se un file non è più necessario, apritelo dalla lista e cliccate su **"Delete"**. Vi verrà chiesta una conferma prima dell'eliminazione definitiva.
