# Layout-configuratie

Toegankelijk via **Admin** > **Merch** > **Layout**, in deze sectie kunt u de lay-out van uw site beheren (menu's, logo, footer).

## Top Bar

![image](https://github.com/user-attachments/assets/750a0a59-8959-4413-a2e1-3283bb70b4cb)

Vertegenwoordigt de hoofdmenubalk van uw be-BOP. Het bevat de volgende parameters.

### Brand Name

Een veld om de naam van uw bedrijf te wijzigen
![image](https://github.com/user-attachments/assets/6d5f6255-0691-47b0-8edc-3e1e21839d4d)

### Logo en favicon

Links waarmee u het logo en de favicon van uw be-BOP kunt wijzigen.

- **Logo wijzigen**: Ga naar **pictures**, voeg een afbeelding toe en stel deze in als logo.
- **Favicon wijzigen**: Ga naar **pictures**, voeg een afbeelding toe en stel deze in als favicon.

### Website Title

De titel die wordt weergegeven in het browsertabblad van uw site.

### Website Description

Een beschrijving van uw site. Deze wordt ook gebruikt als beschrijving bij de indexering van uw site.

![image](https://github.com/user-attachments/assets/156514a5-56f0-4e80-b482-c1c623a58151)

### Links

Menu's die u aan uw be-BOP top bar kunt toevoegen.

Standaard:

| Text  | URL      |
| ----- | -------- |
| Blog  | `/blog`  |
| Store | `/store` |
| Admin | `/admin` |

**[Add topbar link]**: een knop om menu's toe te voegen

## Footer

![image](https://github.com/user-attachments/assets/389f4404-f021-4173-8058-06ddcb1802f0)

Vertegenwoordigt de footersectie van uw site. Het bevat de volgende parameters.

### Opties

- [ ] Display "Powered by be-BOP": toont een "Powered by be-BOP" in uw footer.
- [x] Display company identity & company contact: toont de identiteit van uw bedrijf en uw contactgegevens.
- [x] Display main shop informations: toont de informatie van uw hoofdwinkel.

### Links

Maakt links in uw footer mogelijk.
Standaard:

| Text             | URL        |
| ---------------- | ---------- |
| Terms of Service | `/terms`   |
| Privacy Policy   | `/privacy` |

**[Add footer link]**: een knop om andere footerlinks toe te voegen.
