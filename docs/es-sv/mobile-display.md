# Configuración de Visualización Móvil

Accesible a través de Admin > Merch > Layout, esta sección permite cambiar el comportamiento del diseño de su be-BOP en dispositivos móviles.

![image](https://github.com/user-attachments/assets/0939e4d4-6750-4750-844e-1381676d989d)

## Configuración Predeterminada

Por defecto, su be-BOP utiliza esta configuración:

```markdown
meta name="viewport" content="width=1000"
```

Esto le permite mantener el mismo diseño en cualquier dispositivo.

## Viewport Width

![image](https://github.com/user-attachments/assets/054702cf-5a70-444a-8e19-d0b6f23c8d2f)

Permite establecer un valor para el ancho del viewport.

## Device Width

![image](https://github.com/user-attachments/assets/7509aebb-8d6f-4b69-b2c2-d1d034d813e6)

Permite aplicar `content="width=device-width"` a usuarios específicos de su be-BOP.

## Zona CMS Móvil

![image](https://github.com/user-attachments/assets/9cb296a3-d164-48b5-8a39-38f7c31c4edf)

Cuando está marcado, oculta las zonas CMS adicionales (product-top, order-top...) en móvil.
