# Adicionar Ficheiros Descarregáveis a Produtos ou Páginas CMS

Este guia explica como adicionar ficheiros descarregáveis (por exemplo, avisos de produtos ou guias) que são publicamente acessíveis e não protegidos por tokens ou encomendas. Estes ficheiros podem ser ligados diretamente a páginas de produtos ou páginas CMS.

## Passo 1: Aceder à Secção **Files**

1. Inicie sessão no painel de administração.
2. Navegue até à secção **Merch** no menu superior.
3. Clique em **Files** no submenu.

## Passo 2: Adicionar um Novo Ficheiro

![image](https://github.com/user-attachments/assets/3b9ca426-34f1-4905-af05-24f3b16ade14)

1. Na página **List of file with no associated product**, clique em **New file**.

![image](https://github.com/user-attachments/assets/f4f64cd9-bedb-4af9-8626-90178c95b178)

2. Preencha os campos obrigatórios:

   - **Name of the file**: Forneça um nome descritivo (ex.: "Aviso do Produto").
   - **File**: Carregue o ficheiro clicando em **Browse...** e selecionando o ficheiro apropriado do seu computador.

3. Clique no botão **Add** para carregar o ficheiro.

![image](https://github.com/user-attachments/assets/2687f745-4c29-4c85-8c92-6df71c041b91)

4. **Gerir um ficheiro existente:**

   - Depois de o ficheiro ser adicionado, pode vê-lo na lista de ficheiros.
   - Clique num ficheiro para editar o seu nome, obter um link ou eliminá-lo.
   - **Link permanente:**
      Depois de o ficheiro ser adicionado, é gerado um link permanente válido. Pode copiá-lo clicando no ícone "Copiar" junto ao link.
     Exemplo de link:
      `http://localhost:5173/digital-file/raw/<nome-ficheiro>?key=<chave-única>`
     Pode usar este link nas suas páginas CMS para partilhar os seus ficheiros.
   - **Descarregar diretamente:**
     Os utilizadores podem clicar no botão **"Download file"** para aceder imediatamente ao ficheiro.
   - **Eliminar um ficheiro**
     Se um ficheiro já não for necessário, abra-o a partir da lista e clique em **"Delete"**. Será pedida uma confirmação antes da eliminação permanente.
