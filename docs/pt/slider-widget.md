# Documentação do Widget Slider

Disponível via **Admin** > **Widgets** > **Slider**, esta interface permite criar um slider (carrossel de imagens) usado para apresentar conteúdo visual atrativo no seu be-BOP.

![image](https://github.com/user-attachments/assets/b738d8ae-edd8-4d0f-b33c-72ea8f2311bd)

---

## Adicionar um Slider

Para adicionar um slider, clique em **Add slider**.

![image](https://github.com/user-attachments/assets/75483ff8-032d-487c-92e7-4b38068d1d8c)

### 1. **Slider Title**

- **Descrição**: O título do slider. Este texto pode ser usado como referência para identificar o slider no sistema.
- **Utilização**: Opcional, mas útil para organizar e gerir sliders.

### 2. **Slug**

- **Descrição**: Identificador único para o slider, usado em URLs ou como chave de acesso.
- **Restrições**:
  - Só pode conter letras minúsculas, números e hífens (`-`).

### 3. **Picture**

- **Descrição**: Imagem principal a exibir no slider.
- **Funcionalidade**:

  - Um campo para procurar e carregar um ficheiro do seu computador.
  - Apenas aceita formatos de imagem (JPG, PNG, etc.).

![image](https://github.com/user-attachments/assets/1cf9c9c4-4e19-4093-b5cb-1b1d88ffaf88)

## Editar um Slider

Quando clica num título de slider, pode editá-lo adicionando novas imagens.

![image](https://github.com/user-attachments/assets/226847f0-3c05-4b8a-8620-8a42015444f6)

### 1. **Add Picture**

Clicar em **Add picture** levá-lo-á a uma nova página para associar imagens ao seu slider.

![image](https://github.com/user-attachments/assets/dee5e891-e9d6-4849-b04c-926a33ebbd6b)

Cada slide pode ser associado a um URL que abrirá quando clicar no slide. E uma opção **Open in new tab** para abrir o URL numa nova janela.

![image](https://github.com/user-attachments/assets/a57e06b1-6a64-4311-a9c3-5458cfcda1bc)

## Integração CMS

Para integrar o seu slider numa zona ou página CMS, adicione-o da seguinte forma: `[Slider=slug]`.

![image](https://github.com/user-attachments/assets/363bea42-da42-4364-9099-3d8360603c3a)

E o seu slider será exibido aos seus utilizadores da seguinte forma.
