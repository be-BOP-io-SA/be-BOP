# Adding Downloadable Files to Products or CMS Pages

This guide explains how to add downloadable files (for example, product notices or guides) that are publicly accessible and not protected by tokens or orders. These files can be linked directly to product pages or CMS pages.

## Step 1: Access the **Files** Section

1. Log in to the admin panel.
2. Navigate to the **Merch** section in the top menu.
3. Click on **Files** in the submenu.

## Step 2: Add a New File

![image](https://github.com/user-attachments/assets/3b9ca426-34f1-4905-af05-24f3b16ade14)

1. On the **List of file with no associated product** page, click on **New file**.

![image](https://github.com/user-attachments/assets/f4f64cd9-bedb-4af9-8626-90178c95b178)

2. Fill in the required fields:

   - **Name of the file**: Provide a descriptive name (e.g., "Product Notice").
   - **File**: Upload the file by clicking **Browse...** and selecting the appropriate file from your computer.

3. Click the **Add** button to upload the file.

![image](https://github.com/user-attachments/assets/2687f745-4c29-4c85-8c92-6df71c041b91)

4. **Managing an existing file:**

   - Once the file is added, you can view it in the file list.
   - Click on a file to edit its name, get a link, or delete it.
   - **Permanent link:**
      Once the file is added, a valid permanent link is generated. You can copy it by clicking the "Copy" icon next to the link.
     Example link:
      `http://localhost:5173/digital-file/raw/<file-name>?key=<unique-key>`
     You can use this link in your CMS pages to share your files.
   - **Download directly:**
     Users can click the **"Download file"** button to immediately access the file.
   - **Delete a file**
     If a file is no longer needed, open it from the list and click **"Delete"**. A confirmation will be requested before permanent deletion.
