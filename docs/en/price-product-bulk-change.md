# Bulk Product Price Configuration

Accessible via Admin > Merch > Products **Products price**, this section allows you to bulk change the prices of available products.

![image](https://github.com/user-attachments/assets/e13749db-afbe-4196-8ba8-170e95a4f186)

## Form Fields

For each product, you can change:

**Price amount**: The current price of the product.

**Price currency**: The current currency of the product.
