# Documentazione Gestione Temi

Accessibile tramite **Admin** > **Merch** > **Theme**, questa interfaccia consente di gestire i temi sul vostro be-BOP. Permette di:

- Modificare le famiglie di font e i colori tramite l'interfaccia admin.
- Creare diversi stili, importarli/esportarli e passare facilmente dall'uno all'altro tramite l'interfaccia admin.

  ![image](https://github.com/user-attachments/assets/a9d65a4b-24fa-438c-81ae-b50f12bfaaf4)

## Aggiungere un Tema

Per aggiungere un tema, fare clic su **Create a new theme**.

![image](https://github.com/user-attachments/assets/8e732d4e-aabd-4680-b043-8e1907948401)
