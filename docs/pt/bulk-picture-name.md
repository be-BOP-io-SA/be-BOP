# Configuração de Renomeação em Massa de Imagens

Acessível via Admin > Merch > Pictures **Bulk picture name editor**, esta secção permite renomear em massa as imagens guardadas no armazenamento S3 do seu be-BOP.

![image](https://github.com/user-attachments/assets/50efc726-cc53-4a45-be97-18ca9d4ac149)

## Campos do Formulário

O formulário contém os seguintes campos:

**Current slug**: O identificador da imagem (não editável).

**Name**: O nome atual da imagem.
