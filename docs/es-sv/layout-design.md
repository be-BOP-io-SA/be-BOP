# Configuración de diseño

Accesible a través de **Admin** > **Merch** > **Layout**, esta sección te permite gestionar el diseño de tu sitio (menús, logo, pie de página).

## Barra superior

![image](https://github.com/user-attachments/assets/750a0a59-8959-4413-a2e1-3283bb70b4cb)

Representa la barra de menú principal de tu be-BOP. Contiene las siguientes configuraciones.

### Nombre de la marca

Un campo para cambiar el nombre de tu empresa.
![image](https://github.com/user-attachments/assets/6d5f6255-0691-47b0-8edc-3e1e21839d4d)

### Logo y Favicon

Enlaces que te permiten cambiar el logo y el favicon de tu be-BOP.

- **Cambiar el logo**: Ve a **pictures**, agrega una imagen y establécela como logo.
- **Cambiar el favicon**: Ve a **pictures**, agrega una imagen y establécela como favicon.

### Título del sitio web

El título que se mostrará en la pestaña del navegador de tu sitio.

### Descripción del sitio web

Una descripción de tu sitio. También se usa como descripción al indexar tu sitio para SEO.

![image](https://github.com/user-attachments/assets/156514a5-56f0-4e80-b482-c1c623a58151)

### Enlaces

Menús que puedes agregar a la barra superior de tu be-BOP.

Por defecto:

| Texto | URL      |
| ----- | -------- |
| Blog  | `/blog`  |
| Store | `/store` |
| Admin | `/admin` |

**[Add topbar link]**: Un botón para agregar más menús.

## Pie de página

![image](https://github.com/user-attachments/assets/389f4404-f021-4173-8058-06ddcb1802f0)

Representa la sección del pie de página de tu sitio. Contiene las siguientes configuraciones.

### Opciones

- [ ] Display "Powered by be-BOP": Muestra un "Powered by be-BOP" en tu pie de página.
- [x] Display company identity & company contact: Muestra la identidad y la información de contacto de tu empresa.
- [x] Display main shop informations: Muestra la información principal de tu tienda.

### Enlaces

Enlaces en tu pie de página.
Por defecto:

| Texto            | URL        |
| ---------------- | ---------- |
| Terms of Service | `/terms`   |
| Privacy Policy   | `/privacy` |

**[Add footer link]**: Un botón para agregar más enlaces al pie de página.
