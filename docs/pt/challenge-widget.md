# Documentação do Widget Challenge

Acessível via **Admin** > **Widgets** > **Challenge**, os widgets de desafio podem ser usados no seu be-BOP para acompanhar desafios em vendas, suporte, etc. Podem ser integrados em zonas ou páginas CMS.

![image](https://github.com/user-attachments/assets/88bfde44-acee-40e9-8b81-4bb278072065)

## Adicionar um Desafio

Para adicionar um desafio, clique em **Add challenge**.

![image](https://github.com/user-attachments/assets/5797b4ca-f7d7-4fa0-9f5e-3fe7e4349083)

### 1. **Challenge name**

O nome do desafio. Um título curto e descritivo que resume o objetivo do desafio.

### 2. **Mode**

O modo do desafio, que pode ser um número de itens vendidos (**TotalProducts**) ou um valor em dinheiro a recolher (**MoneyAmount**).

### 3. **Goal**

O objetivo a atingir para o desafio. Acompanhado por um campo **currency** se o modo for **MoneyAmount**, representando a moeda.

### 4. **Beginning date**

A data de início do desafio.

### 5. **Ending date**

O prazo para completar o desafio.

### 6. **Products**

Uma lista de produtos cujas vendas contribuem para o progresso do desafio.

## Integração CMS

Para integrar um desafio numa zona ou página CMS, adicione-o da seguinte forma: `[Challenge=slug]`.

![image](https://github.com/user-attachments/assets/25361bb3-1f69-477e-89d4-10a2abcf7779)

E o seu desafio será exibido aos seus utilizadores da seguinte forma.
