# Configuração de Preços de Produtos em Massa

Acessível via Admin > Merch > Products **Products price**, esta secção permite alterar em massa os preços dos produtos disponíveis.

![image](https://github.com/user-attachments/assets/e13749db-afbe-4196-8ba8-170e95a4f186)

## Campos do Formulário

Para cada produto, pode alterar:

**Price amount**: O preço atual do produto.

**Price currency**: A moeda atual do produto.
