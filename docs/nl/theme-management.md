# Themabeheer Documentatie

Toegankelijk via **Admin** > **Merch** > **Theme**, deze interface stelt u in staat om thema's te beheren op uw be-BOP. Het biedt:

- De winkeleigenaar de mogelijkheid om lettertypefamilies en kleuren aan te passen via de admin UI.
- De eigenaar de mogelijkheid om verschillende stijlen te creëren, te importeren/exporteren en eenvoudig tussen stijlen te schakelen via de admin UI.

  ![image](https://github.com/user-attachments/assets/a9d65a4b-24fa-438c-81ae-b50f12bfaaf4)

## Een Thema Toevoegen

Om een thema toe te voegen, klik op **Create a new theme**.

![image](https://github.com/user-attachments/assets/8e732d4e-aabd-4680-b043-8e1907948401)
