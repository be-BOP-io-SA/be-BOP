# Challenge-Widget-Dokumentation

Zugänglich über **Admin** > **Widgets** > **Challenge**, können Challenge-Widgets in Ihrem be-BOP verwendet werden, um Herausforderungen bei Verkäufen, Unterstützung usw. zu verfolgen. Sie können in CMS-Zonen oder -Seiten integriert werden.

![image](https://github.com/user-attachments/assets/88bfde44-acee-40e9-8b81-4bb278072065)

## Eine Challenge hinzufügen

Um eine Challenge hinzuzufügen, klicken Sie auf **Add challenge**.

![image](https://github.com/user-attachments/assets/5797b4ca-f7d7-4fa0-9f5e-3fe7e4349083)

### 1. **Challenge name**

Der Name der Challenge. Ein kurzer, beschreibender Titel, der das Ziel der Challenge zusammenfasst.

### 2. **Mode**

Der Challenge-Modus, der eine Anzahl verkaufter Artikel (**TotalProducts**) oder ein zu erreichender Geldbetrag (**MoneyAmount**) sein kann.

### 3. **Goal**

Das zu erreichende Ziel der Challenge. Begleitet von einem **currency**-Feld, wenn der Modus **MoneyAmount** ist, das die Währung repräsentiert.

### 4. **Beginning date**

Das Startdatum der Challenge.

### 5. **Ending date**

Die Frist zum Abschluss der Challenge.

### 6. **Products**

Eine Liste von Produkten, deren Verkäufe zum Fortschritt der Challenge beitragen.

## CMS-Integration

Um eine Challenge in eine CMS-Zone oder -Seite zu integrieren, fügen Sie sie wie folgt hinzu: `[Challenge=slug]`.

![image](https://github.com/user-attachments/assets/25361bb3-1f69-477e-89d4-10a2abcf7779)
