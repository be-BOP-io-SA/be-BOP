# Layout Configuration

Accessible via **Admin** > **Merch** > **Layout**, this section allows you to manage the layout of your site (menus, logo, footer).

## Top Bar

![image](https://github.com/user-attachments/assets/750a0a59-8959-4413-a2e1-3283bb70b4cb)

Represents the main menu bar of your be-BOP. It contains the following settings.

### Brand Name

A field to change your company name.
![image](https://github.com/user-attachments/assets/6d5f6255-0691-47b0-8edc-3e1e21839d4d)

### Logo and Favicon

Links that allow you to change the logo and favicon of your be-BOP.

- **Change the logo**: Go to **pictures**, add an image, and set it as the logo.
- **Change the favicon**: Go to **pictures**, add an image, and set it as the favicon.

### Website Title

The title that will be displayed in the browser tab of your site.

### Website Description

A description of your site. Also used for the description when indexing your site for SEO.

![image](https://github.com/user-attachments/assets/156514a5-56f0-4e80-b482-c1c623a58151)

### Links

Menus you can add to your be-BOP top bar.

By default:

| Text  | URL      |
| ----- | -------- |
| Blog  | `/blog`  |
| Store | `/store` |
| Admin | `/admin` |

**[Add topbar link]**: A button to add more menus.

## Nav Bar

![image](https://github.com/user-attachments/assets/b5e808fe-3b97-4fa2-a64f-dfce02e8c7d7)

Represents the secondary navigation of your site. It contains the following settings.

### Links

Submenus of your be-BOP.
By default:

| Text       | URL           |
| ---------- | ------------- |
| Challenges | `/challenges` |
| Rewards    | `/rewards`    |
| Orders     | `/orders`     |

**[Add navbar link]**: A button to add more submenus.

---

## Footer

![image](https://github.com/user-attachments/assets/389f4404-f021-4173-8058-06ddcb1802f0)

Represents the footer section of your site. It contains the following settings.

### Options

- [ ] Display "Powered by be-BOP": Displays a "Powered by be-BOP" on your footer.
- [x] Display company identity & company contact: Displays your company identity and contact information.
- [x] Display main shop informations: Displays your main shop information.

### Links

Links on your footer.
By default:

| Text             | URL        |
| ---------------- | ---------- |
| Terms of Service | `/terms`   |
| Privacy Policy   | `/privacy` |

**[Add footer link]**: A button to add more footer links.

### Social Network Icons

Displays your social network logos.
