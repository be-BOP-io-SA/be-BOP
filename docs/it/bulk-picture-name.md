# Configurazione Bulk Picture Name

Accessibile su Admin > Merch > Pictures **Bulk picture name editor**, questa sezione permette di modificare in massa i nomi delle immagini salvate nel vostro S3 be-BOP.

![image](https://github.com/user-attachments/assets/50efc726-cc53-4a45-be97-18ca9d4ac149)

## Campi del modulo

Il modulo contiene i seguenti campi:

**Current slug**: L'identificativo dell'immagine (non modificabile).

**Name**: Il nome attuale dell'immagine.
