# Slider-Widget-Dokumentation

Verfügbar über **Admin** > **Widgets** > **Slider**, diese Oberfläche ermöglicht es Ihnen, einen Slider (Bildkarussell) zu erstellen, der zur Präsentation attraktiver visueller Inhalte in Ihrem be-BOP verwendet wird.

![image](https://github.com/user-attachments/assets/b738d8ae-edd8-4d0f-b33c-72ea8f2311bd)

---

## Einen Slider hinzufügen

Um einen Slider hinzuzufügen, klicken Sie auf **Add slider**.

![image](https://github.com/user-attachments/assets/75483ff8-032d-487c-92e7-4b38068d1d8c)

### 1. **Slider Title**

- **Beschreibung**: Der Slider-Titel. Dieser Text kann als Referenz zur Identifikation des Sliders im System verwendet werden.
- **Verwendung**: Optional, aber nützlich zum Organisieren und Verwalten von Slidern.

### 2. **Slug**

- **Beschreibung**: Eindeutiger Bezeichner für den Slider, verwendet in URLs oder als Zugriffsschlüssel.
- **Einschränkungen**:
  - Darf nur Kleinbuchstaben, Zahlen und Bindestriche (`-`) enthalten.

### 3. **Picture**

- **Beschreibung**: Hauptbild, das im Slider angezeigt wird.
- **Funktionalität**:

  - Ein Feld zum Durchsuchen und Hochladen einer Datei von Ihrem Computer.
  - Akzeptiert nur Bildformate (JPG, PNG usw.).

![image](https://github.com/user-attachments/assets/1cf9c9c4-4e19-4093-b5cb-1b1d88ffaf88)

## Einen Slider bearbeiten

Wenn Sie auf einen Slider-Titel klicken, können Sie ihn bearbeiten, indem Sie neue Bilder hinzufügen.

![image](https://github.com/user-attachments/assets/226847f0-3c05-4b8a-8620-8a42015444f6)

### 1. **Add Picture**

Durch Klicken auf **Add picture** gelangen Sie zu einer neuen Seite, um Bilder mit Ihrem Slider zu verknüpfen.

![image](https://github.com/user-attachments/assets/dee5e891-e9d6-4849-b04c-926a33ebbd6b)

Jede Folie kann mit einer URL verknüpft werden, die beim Klicken auf die Folie geöffnet wird. Und eine Option **Open in new tab**, um die URL in einem neuen Fenster zu öffnen.

![image](https://github.com/user-attachments/assets/a57e06b1-6a64-4311-a9c3-5458cfcda1bc)

## CMS-Integration

Um Ihren Slider in eine CMS-Zone oder -Seite zu integrieren, fügen Sie ihn wie folgt hinzu: `[Slider=slug]`.

![image](https://github.com/user-attachments/assets/363bea42-da42-4364-9099-3d8360603c3a)

Und Ihr Slider wird Ihren Benutzern wie folgt angezeigt.
